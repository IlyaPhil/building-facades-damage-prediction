# Оценка повреждений фасадов зданий для графовых нейронных сетей

Проект посвящен разработке пайплайна локализации и оценки степени повреждений фасадов зданий. Полученные признаки в виде вектора повреждений фасада предназначены для последующего использования в графовых нейронных сетях (GNN) с целью прогнозирования распространения дефектов в городской застройке.

## Описание проекта

В рамках данной части дипломной работы реализован следующий алгоритм:

* Сбор визуальных данных (фото фасадов зданий)
* Разметка выборочных участков и обучение модели семантической сегментации (SegFormer) для выявления различных типов повреждений
* Обучение модели нейросетевого ранжирования (RankNet) на парах фасадов для калибровки итогового показателя
* Формирование вектора признаков (Damage Index) для последующей интеграции в GNN

## Использование репозитория

Управление зависимостями осуществляется при помощи uv. 
Для установки и активации окружения выполнить:

```bash
uv sync
source .venv/bin/activate
```

Для версионирования и загрузки данных используется DVC (удаленное хранилище настроено на S3):

```bash
dvc pull
```

Обучение моделей поддерживается с интеграцией wandb.
